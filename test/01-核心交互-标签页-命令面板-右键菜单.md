# 核心交互测试：window_tab / command_palette / right_click_menu / preferences

这个文件用来测试插件框架是否真正启动，而不是只测试 Markdown 渲染。

## 应测插件

| 插件 | 操作 | 期望 |
|---|---|---|
| window_tab | 连续打开本目录 10 个文件 | 顶部出现多个标签页，切换后不消失，不遮挡 macOS 文件名 |
| command_palette | 按 `Ctrl+Shift+P` | 出现命令面板，可搜索插件动作 |
| right_click_menu | 在正文区域右键 | 出现插件分组菜单 |
| preferences | 右键菜单或命令面板打开偏好设置 | 可查看和修改插件配置 |
| hotkeys | 打开热键面板 | 能看到自定义热键列表 |
